Zenray uses an ESP 32 chip to connect to the internet and control the MAX72xx LED lights. This enables the device to be managed via a web interface, giving you control over your therapy session.

How It Works

Zenray is built around two main circuits, one to control the MAX72xx and one to control the LED lights. The web interface is hosted on the ESP 32 chip itself, making it easy for users to manage their therapy sessions.
